# Traditional English Cottage - Build Logs

This folder contains the detailed build documentation for the Traditional English Cottage project.

## Build Sequence

1. **[Architectural Research & Planning](001-planning.md)** - Initial inspiration, 2D to 3D challenge, architectural research, and scale selection
2. **[Cottage Construction](002-cottage-construction.md)** - Wall construction, brick application, flint texturing, windows, doors, and roof
3. **[Base & Landscaping](003-base-landscaping.md)** - Layout planning, tarmac road, gravel paths, lawn, and fence construction
4. **[Plants & Garden](004-plants-garden.md)** - Plant planning, copper wire tree construction, and garden flowers
5. **[Finalisation & Details](005-finalisation.md)** - Final plant installation and gallery photography

## Project Overview

- **Scale:** 1:100
- **Duration:** Late June - October 3, 2025 (~3.5 months)
- **Estimated Hours:** 150-200 hours
- **Base Size:** 20cm × 25cm
- **Completion Date:** 3 October 2025

## Key Techniques

- Wood frame construction with paper cladding
- Modelling paste flint stone texturing
- Paper strip brick work (2.5mm × 5mm)
- Multi-layer window fabrication (7 layers)
- Crushed slate tarmac and paths
- Static grass lawn application
- Bamboo fence with wire mesh
- Copper wire tree with stencil-cut leaves
