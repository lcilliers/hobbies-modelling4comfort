## Privacy Policy

*Last Updated: December 4, 2025*

### Overview

Models4Comfort is a personal hobby website documenting scale modeling projects. We are committed to protecting your privacy and being transparent about any data collection or usage.

### What We Collect

**We do NOT collect:**
- ❌ Personal information (name, email, phone)
- ❌ Payment information (no e-commerce)
- ❌ Tracking cookies for advertising
- ❌ Analytics data through third-party services

**What MAY be collected:**

1. **GitHub Pages Hosting**
   - This site is hosted on GitHub Pages
   - GitHub may collect basic access logs (IP addresses, pages visited, browser type) as part of their hosting service
   - See [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for details

2. **Comment System (if enabled)**
   - If we enable [Utterances](https://utteranc.es/) for comments, it uses GitHub Issues for comment storage
   - You must have a GitHub account to comment
   - Your GitHub username and comment content will be publicly visible
   - Comments are stored in the public GitHub repository
   - No tracking cookies or personal data beyond GitHub's own policies

### Cookies

**We do not set cookies.** 

However:
- GitHub Pages hosting may set cookies as part of their service
- If you're logged into GitHub and use the comment system, GitHub sets cookies per their policies

### Third-Party Links

This site contains links to external websites (stock photo sites, manufacturer sites, reference materials, etc.). We are not responsible for the privacy practices of these external sites. Please review their privacy policies.

### User Comments

**If comment system is enabled:**

- Comments are stored publicly on GitHub Issues
- Your GitHub username, avatar, and comment text are visible to all visitors
- You control your GitHub account and can delete your comments via GitHub
- Comments may be moderated to remove spam or inappropriate content
- By commenting, you agree to GitHub's [Terms of Service](https://docs.github.com/en/site-policy/github-terms/github-terms-of-service)

**We will never:**
- Sell or share comment data
- Use comments for marketing
- Contact you unsolicited based on comments

### Contact Forms

**Currently not available.** GitHub Pages is a static hosting platform and does not support server-side form processing. 

If we implement a contact form in the future, we will:
- Use a privacy-respecting third-party service (like Formspree or Netlify Forms)
- Update this privacy policy with details
- Only collect information you explicitly provide
- Never sell or share your contact information

### Children's Privacy

This site is not directed at children under 13. We do not knowingly collect information from children. If comments are enabled, GitHub's age requirements apply.

### Data Security

Since we collect minimal to no personal data:
- No personal data is stored on our servers
- All content is publicly visible (as intended for a public blog)
- GitHub provides security for their platform and any stored comments

### Your Rights

You have the right to:
- Access any public comments you've made (via GitHub)
- Delete your comments (through GitHub Issues)
- Request removal of attributed images (contact via GitHub repository)

### Changes to This Policy

We may update this privacy policy as the site evolves (e.g., if we add features like comments, contact forms, or analytics). Changes will be posted on this page with an updated date.

### International Users

This site is operated from the United Kingdom. If you access this site from outside the UK, your information may be transferred to, stored, and processed in countries where GitHub operates.

### GDPR Compliance (EU/UK)

For visitors from the EU/UK:
- We do not process personal data beyond what GitHub Pages collects for hosting
- GitHub is GDPR compliant - see their [GDPR information](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement#githubs-global-privacy-practices)
- If comments are enabled, your GitHub account information is processed with your consent when you comment

### Contact

For questions about this privacy policy or image credits:
- **GitHub Repository:** [github.com/lcilliers/hobbies-modelling4comfort](https://github.com/lcilliers/hobbies-modelling4comfort)
- **Open an Issue:** Report concerns via GitHub Issues
