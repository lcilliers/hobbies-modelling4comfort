## Join Our Community

Welcome to Modelling4Comfort! We're excited to have you join our community of modelling enthusiasts.

## Registration Options

Since this is a static website hosted on GitHub Pages, we use third-party services for user interaction and registration. Here are the ways you can participate:

### 1. Disqus Comments

We use [Disqus](https://disqus.com) for our comment system. To participate in discussions:

1. Visit any blog post
2. Scroll to the comments section
3. Sign in with your Disqus account (or create one for free)
4. You can also sign in using Google, Facebook, or Twitter

Benefits of registering with Disqus:
- Comment on any post
- Receive notifications for replies
- Build your reputation across sites using Disqus
- Follow other commenters

### 2. GitHub Discussions

For more in-depth discussions and contributions:

1. Create a [GitHub account](https://github.com/join) if you don't have one
2. Visit our [repository](https://github.com/lcilliers/hobbies-modelling4comfort)
3. Use GitHub Issues to report bugs or suggest features
4. Submit pull requests to contribute content

### 3. Newsletter (Coming Soon)

We're working on setting up a newsletter for:
- Weekly updates on new content
- Featured projects from the community
- Tips and tricks from experienced modellers

## Content Submission

Want to contribute your own content?

### Submit Photos

1. Fork our [GitHub repository](https://github.com/lcilliers/hobbies-modelling4comfort)
2. Add your images to the `assets/images/` folder
3. Create or edit a markdown file to include your images
4. Submit a pull request

### Submit Videos

1. Upload your video to YouTube
2. Fork our repository
3. Add your video using the video include in a markdown file
4. Submit a pull request

### Write a Blog Post

1. Fork our repository
2. Create a new file in `_posts/` with the format: `YYYY-MM-DD-title.md`
3. Write your post using markdown
4. Include the required front matter (layout, title, date)
5. Submit a pull request

## Community Guidelines

To maintain a welcoming environment, please:

- Be respectful and constructive in comments
- Give credit when sharing others' work
- Keep discussions relevant to modelling
- Report any inappropriate content

## Need Help?

If you have any questions about registration or participation:

- Leave a comment on any blog post
- Open an issue on our [GitHub repository](https://github.com/lcilliers/hobbies-modelling4comfort)
- Check our [About page](/about/) for more contact information

We look forward to seeing you in our community!
